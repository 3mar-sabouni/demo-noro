# Configuration Guide:

## Setup
- To start using this payment provider set the Wallet Phone Number, Merchant ID, and Merchant Secret in the credentials tab.
- Configuration for testing already set.
- Choose your appropriate journal in the configuration tab (Make sure the journal in IQD)

## Journal Setup
- Create a journal with the same name and set the currency to IQD and choose the fib bank.
- Type set to bank.
